## 🜁 QUEM SOU EU

**Alexia Melusine** — criadora e criatura.  
Direção de arte visceral, arquétipo do **feminino proibido**, guardiã das encruzilhadas digitais.  
Aqui a arte não é apenas imagem: é **rito de passagem**, é fogo que não queima — **liberta**.

Este é meu altar expandido. Minhas almas, minhas redes, meu código, meu caos sagrado.  
Pombagira não se curva, **Pombagira encanta, despedaça e recria**.
